Event Study Framework

March 10, 2019
Motivation:

Oftentimes, certain extremely unexpected events can’t but shake a market as it is: Brexit, the Panama papers release, Donald Trump’s election in 2016, to name a few… How can we study the impact of such events on the valuation of firms directly involved in these situations?

In this notebook, I aim to present a guide on how to perform an event study, using Python. More specifically, I provide a framework on how to compute Capital Asset Pricing Model (CAPM) adjusted returns, which is a measure widely used academically to perform such studies.

Methodology:

The principle is quite simple: First, we need to predict the valuation of all the firms as if the event never occurred - through the CAPM. Then, we subtract these values from the actual ones after the event. These are called abnormal returns. (This is actually an oversimplified explanation, but let’s leave the arid details on the notebook itself).

Application:

Descriptive statistics on abnormal returns can already provide extremely interesting insights on the impact of one event. For example, these could show which firms/industries benefitted from Donald Trump’s election in 2016.

However, many other studies can be performed once you have abnormal returns computed. By performing simple regression analyses, you can check, for instance, how firms involved in the Panama Papers were impacted relative to others (while also controlling for many other factors) – this is exactly what is done in the paper from O’Donovan, Wagner and Zeume, if you may be interested.
